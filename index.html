Remedy Finder

Ruby On Rails

This is my capstone project i did for Actualize Coding Bootcamp graduation.
The idea behind this app is to help people find natural Medecin alternatives to all the chemicaly manufactured meds. i came up with this idea because I've always wondered what do average people in different countries use to heal common illnesses and every time i try to search i wouldn't find a source that feels homemade with a straight foreward recipe.

This App Allows the user to sign up and log in, Post a remedy to the site and if the user find one he or she likes the user can comment on the remedy, add it to their favorite list, follow the OP to read more as the followed user posts new remedies.

There is also a map that filters posts by region which is really cool for curiosity. It gives the user the ability to see all the remedies by regions.
